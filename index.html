Beginner Blog (Static)
Node.js Express

GitHub Pages Friendly Preview

The real project runs on Node.js/Express. This static file exists so GitHub Pages can host a quick preview explaining what the backend does. Clone the repo and run the server to see the dynamic blog.

How to Run Locally

  1. Install Node.js (v18+)
  2. Run npm install
  3. Run npm run dev for nodemon or npm start for plain node

What You Get

  • Express server with middleware
  • EJS views + partials
  • In-memory blog data and routes
  • Static assets served from public/
