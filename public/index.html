Static Entry Point

GitHub Pages only serves static files, so this page exists to describe the Express project. Clone the repo and run the Node server to see the real blog experience on /blogs.

Command Cheatsheet

  • npm install
  • npm run dev for nodemon
  • npm start for production

Route Map

  • GET /blogs list posts
  • POST /blogs create new post
  • GET /blogs/:id single post
  • DELETE /blogs/:id remove a post
